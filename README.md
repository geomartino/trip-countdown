# trip-countdown

Dépôt multi-décomptes pour diverses aventures. Le fichier actif est toujours **`index.html`** afin que la publication via GitHub Pages fonctionne automatiquement. Les décomptes passés sont renommés et conservés en archives dans le dépôt.

## Fonctionnement

- `index.html` — décompte actif (publié sur GitHub Pages)
- Les anciens décomptes sont renommés (ex. `mauricie-2026.html`) et conservés comme archives
